PROMPT UP — Схема базы данных

3 таблицы · 2 связи · SQLite / MSSQL

PK — первичный ключ
FK — внешний ключ
UK — уникальный
users
id varchar(36) PK uuid4, авто default
username varchar(255) UK NOT NULL
email varchar(255) UK NOT NULL
hashed_password varchar(255) bcrypt NOT NULL
is_active boolean default: true
created_at datetime now(UTC)
user_profiles
id varchar(36) PK uuid4, авто default
user_id varchar(36) FK+UK → users.id NOT NULL
level varchar(255) newbie | intermediate | advanced
sphere varchar(255) сфера деятельности
goals varchar(255) цели обучения
profiler_done boolean default: false
tutor_introduced boolean default: false
current_module_id integer 1–6, nullable
total_score integer default: 0
created_at datetime now(UTC)
module_progress
id varchar(36) PK uuid4, авто default
user_id varchar(36) FK → users.id NOT NULL
module_id integer 1–6 NOT NULL
score integer макс. 50 = завершён default: 0
count integer кол-во попыток default: 0
Связи
users 1:1 user_profiles user_id → id Один пользователь имеет один профиль
users 1:N module_progress user_id → id Один пользователь имеет много записей прогресса
Уникальные ограничения
module_progress user_id + module_id uq_user_module Один пользователь — одна запись на модуль
users username unique Уникальный логин
users email unique Уникальный email
user_profiles user_id unique Один профиль на пользователя
Вычисляемые поля (не в БД)
module_progress is_completed score >= 50 Модуль пройден
module_progress avg score / count Средний балл за попытку
